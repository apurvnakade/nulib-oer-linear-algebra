Bijectivity.

To show \(f\) is bijective directly using Definition 1.1.19, we must show that \(f\) is injective and surjective. This is equivalent to showing that for \(y\in Y\) there exists a unique element \(x\in X\) such that \(f(x)=y\text{.}\)

Alternatively, using Theorem 1.1.23 we can show that \(f\) is bijective by providing an inverse function \(f^{-1}\colon Y\rightarrow X\text{.}\)
