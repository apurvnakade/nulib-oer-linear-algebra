Theorem 4.2.17. Reflection is a linear transformation.

Fix an angle \(0\leq \alpha\leq \pi\text{.}\) The reflection \(r_\alpha\colon \R^2\rightarrow\R^2\) is a linear transformation. In fact we have \(r_{\alpha}=T_A\text{,}\) where

\begin{equation*} A=\begin{amatrix}[rr] \cos 2\alpha\amp\sin 2\alpha \\ \sin 2\alpha \amp -\cos 2\alpha \end{amatrix}\text{.} \end{equation*}

Proof.
