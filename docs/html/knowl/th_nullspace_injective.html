Theorem 4.3.15. Nullspace and injectivity.

Let \(T\colon V\rightarrow W\) be a linear transformation. The following are equivalent:

  1. \(T\) is injective (i.e., one-to-one);

  2. \(\NS T=\{\boldzero\}\text{.}\)

In fact, we have
\begin{equation} T(\boldv)=T(\boldv') \iff \boldv'=\boldv+\boldu \text{ for some } \boldu\in \NS T\tag{4.3.1} \end{equation}

Proof.
