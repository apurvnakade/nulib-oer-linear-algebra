Remark 4.3.4. Two-step proof for subspaces.

As with proofs regarding linearity of functions, we can merge conditions (ii)-(iii) of DefinitionÂ 4.3.1 into a single statement about linear combinations, deriving the following two-step method for proving a set \(W\) is a subspace of a vector space \(V\text{:}\)

  1. Show \(\boldzero_V\in W\)

  2. Show that

    \begin{equation*} \boldv_1, \boldv_2\in W\implies c\boldv_1+d\boldv_2\in W\text{,} \end{equation*}

    for all \(c,d\in\R\text{.}\)
