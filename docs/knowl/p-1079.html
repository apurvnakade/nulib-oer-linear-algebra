Define \(r_\alpha\colon \R^2\rightarrow \R^2\) to be the function that takes an input \(P=(x,y)\) and returns the output \(P'=(x',y')\) obtained by reflecting \(P\) through the line \(\ell_\alpha\text{.}\) In more detail: if \(P\) lies on \(\ell_\alpha\text{,}\) then \(P'=P\text{;}\) otherwise, \(P'\) is obtained by drawing the perpendicular through \(\ell_\alpha\) that passes through \(P\) and taking the point on the other side of this line whose distance to \(\ell_\alpha\) is equal to the distance from \(P\) to \(\ell_\alpha\text{.}\)
