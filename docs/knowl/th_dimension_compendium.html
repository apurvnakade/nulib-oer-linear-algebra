Theorem 4.5.18. Dimension theory compendium.

Let \(V\) be a vector space of dimension \(n\text{.}\)

  1. Contract.

    If \(S\) spans \(V\text{,}\) then \(S\) can be contracted to a basis of \(V\text{.}\)

  2. Expand.

    If \(S\) is linearly independent, then \(S\) can be extended to a basis of \(V\text{.}\)

  3. Street smarts.

    If \(S\subseteq V\) and \(\val{S} < n\text{,}\) then \(S\) does not span \(V\text{.}\)

  4. Street smarts.

    If \(S\subseteq V\) and \(\val{S} > n\text{,}\) then \(S\) is linearly dependent.

  5. Street smarts.

    If \(S\subseteq V\) and \(\val{S}=n\text{,}\) then \(S\) is a basis if and only if \(S\) spans if and only if \(S\) is linearly independent.

  6. Dimension of subspaces.

    If \(W\subseteq V\) is a subspace, then

    1. \(\dim W\leq \dim V\text{,}\) and

    2. \(\dim W=\dim V\) if and only if \(W=V\text{.}\)
