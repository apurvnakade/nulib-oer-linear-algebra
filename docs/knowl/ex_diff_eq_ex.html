Example 3.3.19. A differential equation.

Fix an interval \(X\subseteq \R\text{.}\) Let \(S\) be the set of functions of \(C^1(X)\) satisfying the differential equation

\begin{equation} f'=f\text{.}\tag{3.3.1} \end{equation}

Then

\begin{equation*} S=\{f\in C^1(X)\colon f'-f=\boldzero\}=\NS T\text{,} \end{equation*}

where \(T\colon C^1(X)\rightarrow C(X)\) is the linear transformation \(T(f)=f'-f\text{.}\)

Having identified the set of solution to (3.3.1) with the subspace \(\NS T\text{,}\) we can now bring some linear algebra to the table. For example, since \(\NS T\) is closed under vector addition and scalar multiplication, we know that if \(f\) and \(g\) are solutions to (3.3.1), then so is \(cf+dg\) for any \(c,d\in\R\text{.}\)

It is an interesting exercise to try and understand what \(\im T\) means in terms of differential equations. By definition, \(\im T\) is the set of \(g\in C(X)\) such that \(g=T(f)=f'-f\text{,}\) for some \(f\in C^1(X)\text{.}\) In the language of differential equations, this is the set of all continuous functions on \(X\) for which the differential equation

\begin{equation*} f'-f=g \end{equation*}

can be solved for \(f\text{.}\)
