Exercise 1.

Determine which of the following statements are true and which are false.

  1. If \(\displaystyle A\) and \(\displaystyle B\) are square matrices of the same size, then \(\displaystyle (AB)^T = A^TB^T\text{.}\)

  2. If \(\displaystyle A\) and \(\displaystyle B\) are matrices with sizes such that \(\displaystyle AB\) is square, then \(\displaystyle BA\) exists.

  3. The matrices \(\displaystyle A^TA\) and \(\displaystyle AA^T\) are always square, regardless of the size of \(\displaystyle A\text{.}\)

  4. If \(\displaystyle A\) and \(\displaystyle B\) are square matrices of the same size then \(\displaystyle (A+B)^2 = A^2 + 2AB + B^2\text{.}\)

  5. The matrices \(\displaystyle A^TA\) and \(\displaystyle AA^T\) are always defined, regardless of the size of \(\displaystyle A\text{.}\)
