Theorem 6.1.9. Coordinate vector transformation.

Let \(B=(\boldv_1,\boldv_2,\dots, \boldv_n)\) be an ordered basis for the vector space \(V\text{.}\) Define \(T\colon V\rightarrow \R^n\) by \(T(\boldv)=[\boldv]_B\text{.}\)

  1. \(T\) is a linear transformation. We will call \(T\) the coordinate vector map with respect to \(B\).

  2. \(T(\boldv)=T(\boldw)\) if and only if \(\boldv=\boldw\text{.}\) In particular, \(T(\boldv)=(0,0,\dots, 0)\) if and only if \(\boldv=\boldzero_V\text{.}\)

  3. \(\displaystyle \im T=\R^n\)

  4. Given a subset \(S=\{\boldw_1,\boldw_2,\dots, \boldw_r\}\subseteq V\text{,}\) let \(S'=\{[\boldw_1]_B,[\boldw_2]_B,\dots, [\boldw_r]_B\}\subseteq \R^n\text{:}\) i.e., \(S'\) is the set of coordinate vectors of the elements of \(S\text{.}\) We have the following equivalences:

    1. \(S\) spans \(V\) if and only if \(S'\) spans \(\R^n\text{;}\)

    2. \(S\) is linearly independent if and only if \(S'\) is linearly independent.
