\(S = \lbrace a,b,c\rbrace\)\(T = \lbrace a,c,d,e\rbrace\)\(S\cup T\text{,}\) or \(S\text{ "union" } T\text{,}\) is defined as the set of elements that are contained in either \(S\) or \(T\text{.}\) In this case, the answer is \(\lbrace a,b,c,d,e\rbrace\) Remember that even if an element is in both sets, it is only counted once.
