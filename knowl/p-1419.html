It turns out that the null space and image of a linear transformation \(T\) tell us whether \(T\) is injective and/or surjective as a function. Recall from Definition 0.1.19 that a function \(f\colon X\rightarrow Y\) is:

  • injective (or one-to-one) if for all \(x,x'\in X\) we have \(f(x)=f(x')\) implies \(x=x'\text{;}\)

  • surjective (or onto) if for all \(y\in Y\) there is an \(x\in X\) with \(f(x)=y\text{.}\)

As observed in Definition 0.1.19, we can express surjectivity in terms of image as follows: \(f\) is surjective if and only if \(\im f=Y\text{.}\) Returning to our linear transformation \(T\colon V\rightarrow W\text{,}\) it follows immediately that \(T\) is surjective as a function if and only if \(\im T=W\text{.}\) Similarly, Theorem 3.4.13 tells us that \(T\) is injective as a function if and only if \(\NS T=\{\boldzero_V\}\text{.}\)
