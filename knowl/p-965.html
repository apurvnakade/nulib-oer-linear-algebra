Let \(A\) be an \(m\times n\) matrix. Using Gauss-Jordan elimination, we can row reduce \(A\) to matrix \(U\) in reduced row echelon form. Suppose \(A\) is also row equivalent to the matrix \(U'\) in reduced row echelon form. Then \(U\) and \(U'\) are row equivalent, since the row equivalence relation is symmetric and transitive (RemarkÂ 2.4.16). Thus it suffices to show that if \(U\) and \(U'\) are row equivalent matrices in reduced row echelon form, then \(U=U'\text{.}\) We do so by induction on \(n\text{.}\) The base step is trivial, since there is only one \(m\times 1\) matrix in reduced row echelon form.
