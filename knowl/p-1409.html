Let \(T\colon V\rightarrow W\) be a linear transformation, and let \(\boldw\in W\text{.}\) Equation (3.4.3) can be interpreted as follows: if we can find one particular input \(\boldv_p\) satisfying \(T(\boldv_p)=\boldw\text{,}\) then the set \(X_\boldw\) of all inputs \(\boldv\) satisfying \(T(\boldv)=\boldw\) is given by

\begin{equation*} X_\boldw=\{\boldv_p+\boldu\colon \boldu\in \NS T\}\text{.} \end{equation*}

This set \(X\) is not necessarily a subspace. Indeed, if \(\boldw\ne \boldzero_W\text{,}\) then \(\boldzero\notin X\text{,}\) since \(T(\boldzero)=\boldzero_W\text{!}\) Instead, \(X\) is what is called the translate of the subspace \(\NS T\) by the vector \(\boldv_p\text{,}\) and is denoted as \(X=\boldv_p+\NS T\text{.}\) The corollary below is an application to solutions to matrix equations (equivalently, linear systems) obtained by treating the special case where \(T=T_A\) is a matrix transformation associated to an \(m\times n\) matrix \(A\text{.}\)
