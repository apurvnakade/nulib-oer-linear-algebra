Let \(W=\{B\in M_{nn}\colon B^T=-B\}\text{,}\) subspace of skew-symmetric \(n\times n\) matrices. We claim \(\im S=W\text{.}\) As this is a set equality, we prove it by showing the two set inclusions \(\im S\subseteq W\) and \(W\subseteq \im S\text{.}\) (See Basic set properties)
